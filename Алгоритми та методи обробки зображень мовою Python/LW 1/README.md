Програма не зможе виконати усі передбачені функції, якщо не буде створено папку з фалом зображення у каталозі проекту, де розташований файл з вихідним кодом. Налаштувати назву файлу з його розширенням та назву папки в якій він розміщений можна відредагувавши 64 та 66 строчки коду.

Скрипт був проектований з можливістю запуску як з платформи Windows, так і Unix подібних систем, для цього були імпортовані інструменти з модуля “pathlib” та “os”. 

Програма розбита на функції, виклик яких відбувається з 68 по 72 та 57-58 строк коду. Кожна функція має свою строку опису функції. Усі функції, окрім функції “task” проектувалися для гнучкого використання. Функція “task” необхідна для виконання унікальних задач відповідно до 19 варіанту.

Застереження: у коді застосована конструкція “match case” яка була додана з версії Python 3.10, тож скрипт не буде працювати на більш старих версіях мови Python.
